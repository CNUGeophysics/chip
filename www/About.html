geoCHronology Interactive Personal database(CHIP):



This application provides reference, sample, and analysis value data based on the method of analysis and geological age. Seven analysis techniques (Ar-Ar, Cosmogenic, K-Ar, Lu-Hf, OSL, Sm-Nd, U-Pb) and the International Chronostratigraphic Chart (2022/02) are utilized to compile the offered data. This application enables researchers to download isotope and geochronological data , as well as reuse and reprocess the data in accordance with the FAIR principle. This is suitable for college students studying geochemistry, graduate students and researchers researching isotope dating methods, and app developers.


[Search Data]

There are three tabs on the search data page: Reference, Sample, and Map. The Reference tab displays reference and sample information based on the specified analysis method and geological age. The Sample tab presents sample and analysis value data based on the analysis method and geological age. The Map tab shows sample information according to analysis method and geological age when the user selects a marker.

[Submission Manual]

The Submission Manual page provides manuals and templates for users to upload data.

[Submit Data]

The Submit Data page allows users to upload data in 4 steps. The added data is automatically synchronized to CHIP. In addition, the new data provides information in Search Data.
